# Mid-Core Game Profile

## Expected Architecture

- Clear domain modules
- MVC-style boundaries with explicit systems
- State machines or event bus only where justified
- Resource manager
- Save/cloud sync layer
- Numeric schemas

## Default Constraints

- Progression balance
- Economy stability
- Regression control
- Observability
- Performance budget

## Suggested Modules

- Preflight debug capture
- Numeric simulation
- Config validation
- Reward trace
- Regression check plan
- Asset manifest checks
