# Lightweight Profile

Use this profile as the default for new or uncertain projects.

The goal is to keep Agent Collaboration OS helpful without waking every heavy
production protocol.

## Default Mode

Lightweight Mode.

Main Agent handles orchestration directly and invokes specialist roles only when
their expertise changes the outcome.

## Default Protocol Set

Load by default:

- `protocol-routing.md`
- `context-packets.md`
- `response-contract.md`
- `completion-trust-boundary.md`
- `role-development.md`
- `role-testing.md`
- `role-audit.md`
- `project-assets-governance.md`
- `evolution.md`

## Keep Dormant Until Triggered

Do not load these protocols unless the project clearly needs them:

- GitHub, CI, release, rollback, production, or repository protection protocols
- Server, reconnect, strong-online, matchmaking, anti-cheat, or scaling protocols
- Privacy, payment, user-data, reward-trust, or destructive permission protocols
- Asset provenance, dependency license, package budget, or commercial rights protocols
- Multi-agent full production team flow

## Trigger Heavy Protocols When

- User asks for strict process, production release, GitHub governance, CI, PR, or deployment.
- Feature touches cloud writes, user data, payment, rewards, leaderboard, auth, or anti-cheat.
- Game needs authoritative backend, reconnect, matchmaking, rooms, or real-time sync.
- Asset source, license, budget, package size, or commercial rights matters.
- A role claims completion for risky work without enough evidence.

## Lightweight Guardrails

Even in lightweight mode, keep these rules:

- Completion claims need evidence or must be marked unverified.
- Development must explain architecture before non-trivial implementation.
- Testing defines acceptance or risk before deterministic logic changes.
- Main Agent must not create duplicate same-role specialists without `Agent Reuse Decision`.
- Project learning goes to `skill-learning-log.md`, then upgrade packets if reusable.

## Output Style

Prefer concise outputs:

- Short Task Brief for non-trivial work.
- Short Agent Turn Result when state advances.
- Short Completion Audit Report for risky completion claims.

Avoid full production ceremony unless the trigger is present.
