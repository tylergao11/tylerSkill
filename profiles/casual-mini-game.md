# Casual Mini Game Profile

## Expected Architecture

- Simple scene management
- MVC-style separation
- Config-driven numeric data
- Lightweight cloud/service layer when needed
- Minimal state machines

## Default Constraints

- Package size
- Startup time
- Low-end device frame rate
- Asset loading
- Simple save and reward correctness

## Suggested Modules

- Preflight debug capture
- Debug bisection
- Config validation
- Asset manifest checks
- Reward trace
- Cloud function boundary
