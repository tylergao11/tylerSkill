# Templates

Copyable workflow templates live here.

Templates should be short, UTF-8, and stable. If a template becomes specific to a
project, copy it into the consumer project's `docs/project-notes/`,
`docs/handoffs/`, `docs/reviews/`, or `docs/decisions/` folder instead of
editing the shared template or writing runtime files into the skill directory.

Role prompt templates:

- `role-development-prompt.md`
- `role-client-development-prompt.md`
- `role-server-development-prompt.md`
- `role-art-prompt.md`
- `role-testing-prompt.md`
- `role-audit-prompt.md`
